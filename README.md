### 1. View the Landing Page
Open `index.html` in a web browser to see the investor-facing website. Features include:
- Executive summary
- Financial projections
- Competitive advantages
- Team profiles
- Contact form

### 2. Send the One-Pager
Use `marketing/one-pager.md` for cold outreach or initial investor conversations. It's a quick 1-page summary that fits in an email.

### 3. Send the Executive Summary
After initial interest, send `marketing/executive-summary.md` as a PDF. It's a comprehensive 15-page document covering:
- Problem, solution, market opportunity

Use `marketing/investor-pitch-deck.md` to create a PowerPoint/Keynote presentation. The outline includes:
